What is XML?

XML stands for eXtensible Markup Language. It is a markup language used to store and transport data.

How is XML used?

XML is used to structure, store, and transport data in a readable format.

What does XML look like?

Here is an example of XML:

        
            <meeting>
                <organizer>XML</organizer>
                <attendee>is</attendee>
                <subject>very</subject>
                <date>easy</date>
                <time>to</time>
                <location>understand</location>
            </meeting>

What is JavaScript?

JavaScript is a programming language used to create interactive and dynamic web content. It can manipulate and validate data on web pages.

How is JavaScript related to HTML and CSS?

JavaScript, HTML, and CSS are all essential technologies for web development. HTML defines the structure, CSS defines the presentation, and JavaScript adds interactivity and dynamic behavior.

What are some common JavaScript elements, properties, methods, and functions?

  • JavaScript Functions
        
            let foods = ["Pizza", "Sushi", "Tacos", "Ice Cream"];
            console.log(foods[0]); // Outputs Pizza
  • JavaScript Methods
        
            let str = "bye";
            console.log(str.toUpperCase()); // Output: BYE
  • JavaScript Elements
        
            let num = 20;
            console.log(num); // Output: 20
  • JavaScript Properties
        
            let str = "Hello";
