Educational Website For Form Elements

1. Text Input

Please enter your name.

Text inputs are used to collect brief text-based information such as names, email addresses, passwords, or age.

2. Date and time Form control

Please enter today's date.

The time and date form control allows users to input both date and time.

3. Select List

Please select what day it is today.

Dropdown menus allow users to choose from a list of options, making it easy to select one.

4. Scrolling Text Box

Please type in your most favorite childhood memories in the comments box.

Scrolling text box are great for gathering lengthier text, like comments or messages.

5. Radio Buttons

How is your day going?

Radio buttons allow users to select only one option from a range of choices.
